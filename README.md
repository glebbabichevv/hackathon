# RiseOS — Smart City Platform

**AI-панель управленческих решений для города Алматы**

> Хакатон-проект · MVP · 2025

---

## О проекте

**RiseOS** — аналитическая система мониторинга состояния города в реальном времени. Объединяет данные из 10+ внешних источников по 4 ключевым секторам и отвечает на 3 главных управленческих вопроса:

1. **Что происходит** в городе прямо сейчас?
2. **Насколько это критично?**
3. **Какие действия** необходимо предпринять?

---

## Функции системы

### 📊 Мониторинг секторов
| Сектор | Что отслеживается |
|--------|-------------------|
| 🚗 Транспорт | Загруженность дорог, ДТП, пробки, перекрытия, время в пути |
| 🌿 Экология | AQI, PM2.5, PM10, NO₂, O₃, уровень шума, качество воды |
| 🛡️ Безопасность | Инциденты правопорядка, сейсмическая активность, уровень угроз |
| ⚙️ ЖКХ | Нагрузка на электросеть, аварии водоснабжения, вывоз мусора |

### 🤖 AI-компоненты
| Функция | Описание |
|---------|----------|
| **AI-аналитик** | Анализирует все KPI и инциденты, формирует ответы на 3 управленческих вопроса + прогнозы |
| **AI-чат** | Диалоговый интерфейс — отвечает на любые вопросы о состоянии города с учётом текущих данных |
| **Детектор аномалий** | Статистический анализ по Z-score (>2.0) и IQR за последние 24 часа |
| **Движок прогнозов** | Линейная регрессия, горизонт +1ч / +2ч / +4ч для критичных KPI |
| **Движок корреляций** | Умный кросс-доменный анализ — 6 правил с адаптивными порогами (см. ниже) |
| **Генератор инцидентов** | Claude генерирует реалистичные городские инциденты каждые 60 сек |
| **Симуляция кризиса** | Одна кнопка запускает 3 параллельных критических инцидента для демонстрации |

### 🗺️ Карта города
- Интерактивная карта на базе **2GIS MapGL** с реальными слоями:
  - Реальные ДТП и перекрытия (HERE Traffic)
  - Сейсмические события (USGS)
  - Станции качества воздуха (WAQI)
- Маршруты от штаба до инцидента
- Тепловые зоны по типу и критичности
- Поиск адресов через 2GIS Geocoding API

### 📈 Визуализация данных
- KPI-карточки с трендом и пороговыми значениями
- Графики за 24 часа (Area Chart с зонами предупреждений)
- Радар-диаграмма здоровья города по секторам
- Цветовая индикация: 🟢 норма / 🟡 внимание / 🔴 критично
- Live-бегущая строка с реальными данными
- Тост-уведомления о новых инцидентах

### 🔗 Движок корреляций (Smart Correlation Engine)

Система обнаруживает **кросс-доменные риски** — ситуации, которые затрагивают сразу несколько секторов одновременно. В отличие от обычных порогов KPI, движок анализирует **комбинации факторов** и учитывает погоду, время суток и взаимозависимость городских систем.

**Активные правила:**

| # | Правило | Условие срабатывания | Секторы |
|---|---------|----------------------|---------|
| 1 | Каскадный риск ДТП | Дождь + трафик > **82%** + час пик (7–9, 17–20) | Транспорт, Безопасность |
| 2 | Транспортный коллапс к утру | Снегопад ночью (22:00–06:00) | Транспорт, ЖКХ |
| 3 | Рост AQI к утру | AQI > **85** + ветер < **5 км/ч** + ночь | Экология |
| 4 | Риск каскадного отключения | Нагрузка сети > **93%** | ЖКХ, Безопасность |
| 5 | Туман + час пик | Туман + активный час пик | Транспорт |
| 6 | Рост ночных вызовов | Инциденты > **25** ночью | Безопасность |

**Как работает:**
- Каждое правило имеет **уровень уверенности (confidence)** — процент достоверности корреляции на основе исторических данных
- Пороги намеренно **консервативны**: система сигнализирует только при реально аномальных значениях, не при норме
- Критичность (`КРИТ` / `ВНИ`) определяется отдельно внутри каждого правила — например, AQI > 110 или нагрузка > 96% нужна для `КРИТ`
- Результаты отображаются в панели **Корреляции** на правой боковой панели и передаются на карту города

### 👤 Ролевой доступ
При входе выбирается роль, которая фокусирует интерфейс на нужном секторе:
- **Аким** — полный обзор всех секторов
- **Транспорт** — дороги и трафик
- **Экология** — воздух и загрязнения
- **ЖКХ** — инфраструктура и коммунальные сети
- **Безопасность** — правопорядок и угрозы

### 📤 Прочее
- Экспорт HTML-отчёта для печати / PDF
- Голосовое зачитывание инцидентов (Web Speech API, русский язык)
- Адаптивный дизайн (desktop / планшет)

## Интеграции

### Внешние API (реальные данные)
| Сервис | Что предоставляет | Ключ |
|--------|-------------------|------|
| **2GIS MapGL** | Интерактивная карта, маршруты, геокодинг | `VITE_2GIS_KEY` |
| **HERE Traffic API** | ДТП, перекрытия дорог — обновление каждые 2 мин | `VITE_HERE_API_KEY` |
| **WAQI** | Индекс качества воздуха (AQI) по станциям Алматы | `VITE_WAQI_TOKEN` |
| **OpenWeatherMap** | Текущая погода, AQI, 6-часовой прогноз | `VITE_OWM_KEY` |
| **Open-Meteo** | Температура, влажность, ветер, давление — бесплатно | без ключа |
| **Open-Meteo CAMS** | PM2.5, PM10, NO₂, O₃ от Copernicus | без ключа |
| **USGS Earthquake** | Землетрясения M≥2.5 в регионе Алматы | без ключа |
| **OpenSky Network** | Низкоуровневые полёты над городом | без ключа |
| **Nominatim** | Геокодинг адресов (OpenStreetMap) | без ключа |

### AI / LLM
| Сервис | Применение |
|--------|-----------|
| **Anthropic Claude API** | AI-аналитик, AI-чат, генерация инцидентов, анализ на карте |
| **Ollama (локально)** | Оффлайн-режим — llama3.2, mistral, qwen2.5 |

## Быстрый старт

```bash
npm install
cp .env.example .env   # добавить API ключи
npm run dev
```

Файл `.env`:
```
VITE_ANTHROPIC_API_KEY=sk-ant-...
VITE_2GIS_KEY=...
VITE_HERE_API_KEY=...
VITE_WAQI_TOKEN=...
VITE_OWM_KEY=...
```
