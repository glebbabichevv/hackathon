## 🧠 AI ФУНКЦИИ (собственные алгоритмы)

### 🔴 Детектор аномалий (Z-score + IQR)
- Сравнивает текущее значение KPI с историей 24 часов
- Z-score > 2.0 → аномалия с уровнем уверенности (high/medium/low)
- IQR метод как второй фильтр (устойчив к выбросам)
- Визуальная подсветка KPI карточки при аномалии
- **Не внешний API — собственный алгоритм**

### 📈 Прогнозирование (линейная регрессия + OWM)
- Метод наименьших квадратов по истории KPI
- Прогноз на +1ч, +2ч, +4ч для каждого KPI
- R² для оценки качества прогноза
- "Достигнет критического порога через ~X часов"
- Параллельно: официальный 5-дневный прогноз AQI от OpenWeatherMap
- **Два источника прогноза рядом: наш алгоритм vs метеомодель**

### 🔗 Движок корреляций (кросс-доменные правила)
Система автоматически обнаруживает скрытые связи между секторами:
- Дождь + загруженность > 65% + час пик → **риск каскада ДТП (+47%)**
- AQI > 60 + ветер < 8 км/ч + ночь → **прогноз: AQI +30% к утру**
- Нагрузка электросети > 88% → **риск каскадного отключения**
- Снегопад ночью → **прогноз: транспортный коллапс к 08:00**
- Рост ночных вызовов > 15/ч → **усиление патрулирования**
- Каждая корреляция: название + описание + % уверенности + рекомендация

### 💬 AI Анализ города (Ollama + Claude)
Отвечает на три вопроса управленца:
1. **Что происходит** — текущая сводка по всем секторам
2. **Насколько критично** — оценка + наиболее острые проблемы
3. **Что делать** — 3-4 конкретных управленческих действия
4. **Прогнозы** — 3 тезиса на ближайшие 2 часа

Работает на Ollama llama3.2 локально (не нужен интернет, не нужен ключ).
Резервно: Claude API.

### 💬 AI Чат
- Пользователь задаёт любой вопрос о состоянии города
- Ollama/Claude отвечает с учётом текущих KPI и инцидентов
- Контекст: все активные данные дашборда

### 📰 RSS Парсер (Ollama NLP)
- Ollama читает заголовок новости на русском языке
- Определяет: городской инцидент или нет
- Если да: извлекает сектор, severity, локацию, рекомендации
- Nominatim конвертирует адрес в координаты
- **Полностью локальный пайплайн без платных API**

## 🎤 ПИТЧ — ОДНА СТРАНИЦА

**Проблема:** Руководство города получает данные из десятков разрозненных систем. Нет единой картины. Нет предупреждений. Нет рекомендаций.

**Решение:** RiseOS — AI-платформа мониторинга которая:
- Агрегирует **8+ реальных источников данных** (USGS, Open-Meteo, HERE, WAQI, OpenSky, RSS, 2GIS, egov.kz)
- Запускает **3 уровня AI**: локальный LLM (Ollama) + статистические алгоритмы + движок корреляций
- Отвечает на 3 вопроса: **что происходит / насколько критично / что делать**
- Работает **без платных API** для основных функций (Ollama локально)

**Технологии:** React + TypeScript, Leaflet, Ollama llama3.2, Claude API, Open-Meteo, USGS, HERE, WAQI, Nominatim, RSS NLP парсинг

**Целевая аудитория:** Акимат Алматы — руководители управлений транспорта, экологии, ЖКХ, безопасности
